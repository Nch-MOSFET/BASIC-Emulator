# 独自フォーマット仕様について

以下に独自フォーマット（以下本フォーマット）の概要を示します

- 大文字・小文字の区別をなくし、全て大文字として保存
- 既存の文字データとバイナリの関係との互換性をなくし、管理を効率化
- 使用する文字を64種類に限定

以後、使用される文字データに対応するバイナリを`インデックス`と呼びます  

本フォーマットでのインデックスと文字データの対応は以下の表の通りです
